Manual de Instalação

1- 	Baixar e instalar o mongobd e o nodejs em sua maquina;

2- 	Abrir o cmd, acessar o diretorio do mongodb, e executar o comando mongod para inicializar o servido do mongodb.
	O cmd onde o servidor esta rodando NÃO pode ser fechado.
	
3- 	Abrir um segundo cmd, acessar o diretorio do mongodb, e executar o comando mongo para acessar o banco.

4-	No cmd onde o acesso ao banco está rodando execute o comando (use eleicao), para criar o banco.

5- 	Apos executar o (use eleicao) executar os comandos abaixo para criar as collections e adicionar os dados a elas.
	cria collection eleitors e adicionar dados a ela, lembrar de alterar o diretorio de file para o mesmo onde salvou o projeto.

		mongoimport --db eleicao --collection eleitors --type txt --headerline --file c:/dadweb/eleitorado2.txt

	cria collection candidatos e adicionar dados a ela, lembrar de alterar o diretorio de file para o mesmo onde salvou o projeto.

		mongoimport --db eleicao --collection candidatos --type txt --headerline --file c:/dadweb/candidatos2.txt

	Apos executar os dois comando vc pode executar os seguintes comando para verificar se os dados foram inseridos no sistema.

		db.eleitors.find().pretty()
		db.candidatos.find().pretty()

	Apos executar os comandos anteriores o banco já está criado.

6-	Abrir mais um cmd, nele deve acessar o diretorio do projeto e executar o seguinte comando
	
		nodemon server.js

7- 	Abrir um Browser e acessar.
		
		localhost:8080
